# Git Shuffle

A tool for editing Git history: conveniently reordering commits in a branch before publishing or merging.

## Building

```bash
# Quick build
./build.sh

# Manual build
mkdir build
cd build
cmake -DCMAKE_BUILD_TYPE=Release ..
cmake --build .
```

## Requirements

- CMake 3.25+
- Qt6
